# UX Onderzoeksrapportages

POC voor een omgeving die het mogelijk maakt om rapportages van gebruikersonderzoeken simpel in een online formaat aan te bieden.

Momenteel worden onderzoeksrapporten voornamelijk in PDF formaat opgeleverd. Deze wordt op diverse manieren gedeeld (e-mail, intranet, etc.). Bij een eventuele update aan een document bestaat het risico dat er meerdere versies in omloop zijn. Het is bovendien vaak grasduinen om de rapportages te vinden.

Het voorstel is om te kijken of een online format dat simpel te bewerken is en als vervanging kan dienen voor de huidige documentatie. Dit maakt het tevens mogelijk om de onderzoeken daar te plaatsen waar deze relevant zijn. Het delen van onderzoeksrapportages gebeurd door het delen van de betreffende url. Een laatste versie is bovendien altijd gegarandeerd.
